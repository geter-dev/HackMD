# 歡迎來到 HackMD！

歡迎來到 HackMD！👋 這是您可以用來記錄筆記、分享發現、制定專案路線圖、遠端協作等各種需求的可靠工具——一切都基於 Markdown。透過我們的即時協作編輯器，您和團隊可以在無縫、互聯的環境中編輯、分享和討論，隨時隨地都能輕鬆協作。

在本教學中，我們將引導您瞭解 Markdown 的基本知識，以及如何充分利用 HackMD。透過 HackMD，您可以輕鬆地自定義文件，讓您的工作流程更簡化，並與團隊無縫合作。

## 工作空間及側邊欄
每當您登入 HackMD 帳戶時，都會看到一個工作空間。您在 HackMD 所做的一切都會在工作空間中進行。把這裡當成你的家，您可以在這裡建立新的筆記、跳回現有的筆記、變更設定 - 任何您需要的東西，讓您保持有條不紊，輕鬆創作。

無論是您自己或團隊，每個工作空間都由側邊欄和筆記列表組成。
- 筆記列表佔了大部分的頁面。您可以在這裡找到所有筆記。
- 在側邊欄您可以找到... :point_down: 
### 側邊欄
![image](https://hackmd.io/_uploads/Hy9dbdSaC.png =300x)

1. 個人工作空間與團隊工作空間之間切換。
2. 在目前工作空間中進行關鍵字搜尋。
3. 建立新筆記。
4. 免費版當中，垃圾筆記將在三天後永久刪除。
5. 透過標籤篩選工作空間中顯示的筆記（有助您更快找到筆記）。
6. 顯示你收藏的筆記。
7. 最近查看的筆記，包含協作筆記。

>[!Tip]個人 vs 團隊 的側邊欄 
>您的團隊工作空間側邊欄可能會有些不同，這取決於您在特定工作空間中的權限。

## 個人工作空間
您的個人工作空間是您的領域。您可以在這裡記錄每日活動、撰寫學習筆記、集思廣益 -- 任何您想做的事！這是您的空間。只有您才能進入。

您的個人工作空間也是您管理帳戶的地方。

### 帳戶管理

每個個人工作空間都有...

![image](https://hackmd.io/_uploads/rJnajwDWke.png =300x)

1. 帳號設定
2. 個人檔案頁（每個帳號的公開頁面）
3. 查看所有你參與協作的團隊
4. 訂閱相關設定
5. 查看更多：
	- 自訂介面語言
	- HackMD 聯繫方式、填寫回饋
	- 使用條款

## 建立筆記
建立第一篇筆記只花幾秒鐘。

1. 按一下側邊欄頂端的「建立筆記」按鈕。
2. 一旦您的筆記打開，您就可以給它一個標題、添加筆記標籤，並開始寫下想法。

![圖片](https://hackmd.io/_uploads/Sk5DSLCyJx.png =300x)

>[!Tip] 筆記標籤可以幫助您有效組織工作。
>將您的筆記組織成群組，讓您的工作空間井井有條，筆記也容易找到。

迫不及待開始撰寫筆記了嗎？就像你在其他編輯平台一樣，點按工具列按鈕套用文字效果。 粗體、斜體、項目符號和表格都可以一鍵套用。
![圖片](https://hackmd.io/_uploads/r1PRBLAkJg.png =550x)
這裡查看 Markdown 語法作簡表。
![image](https://hackmd.io/_uploads/HkRu_wPWJx.png =300x)

## 從範本建立筆記

為了幫助您開始使用 HackMD，我們製作了許多範本，您可以將其作為筆記的起點。

1. 請在您的工作空間中，點擊 <i class="fa fa-ellipsis-v" aria-hidden="true"></i> 再按一下「從範本新增筆記」
2. 在這裡您可以檢視我們完整的範本清單。這些範本涵蓋許多不同的用途，您可以將其作為起點或靈感來源。
3. 決定範本後，按一下「套用此範本」，就可以將其開啟為一篇新的筆記。
![image](https://hackmd.io/_uploads/rJ767DH6R.png =300x)

#### HackMD 有許多實用的官方範本 :star: 
- 📑 專案 README 文件
- 🛠️ 專案簡報
- 🖋️ 會議記錄
- 🎨 設計規劃

## 插入圖片或GIF
您可以豐富你的筆記，插入專案圖片、線下活動照片、或示範GIF圖檔

上傳影像有兩種主要方式： 
1. 按一下工具列上的圖片圖示，選擇您要插入的圖片。
2. 將檔案拖放到筆記編輯器中。
	![image](https://hackmd.io/_uploads/HJhRuvvbye.png =300x)

我們支援這些圖片和 gif4 格式：PNG、JPG、GIF、BMP 和 TIF。

您可以在圖片上傳完成的連結後方加上尺寸屬性、調整圖片大小，像是 ` =300x` 或 ` =50%`

## 留言
留言是直接在您的筆記中提供或接收回饋最簡單的方式。

有兩種主要的方式留下留言：
1. 在任何筆記中選擇一段文字。
![image](https://hackmd.io/_uploads/SJObQDPbke.png)
2. 點擊「留言圖示」按鈕來查看所有留言。在右側面板上方可以切換 留言／建議修訂 的下拉選單。
![image](https://hackmd.io/_uploads/SJRxyCQFR.png =300x)

除了對筆記留下留言之外，留言面板還可讓您檢視該筆記的所有留言，並與之互動。您甚至可以透過 @ 提到其他已註冊的人，在留言的回覆中標記他們。

想要刪除或解決留言嗎？按一下右上方的三個點展開功能表，就能執行您想要的動作。

![image](https://hackmd.io/_uploads/BJI0EmNDR.png =400x)

您可以隨時重新開啟已解決的留言。按一下頁面上方的留言面板按鈕 `💬`，然後按一下「*標記為 已解決*」按鈕展開原本的留言。從那裡按一下您要重新開啟的留言旁邊的三個點，然後按「撤回解決」。

## 建議修訂
建議修訂功能可讓您對與他人合作的內容提出編輯建議：更正錯字、建議更好的措辭，甚至建立待辦事項清單。

首先，您要確定建議修訂功能已在您的筆記中開啟。只需要...
1. 開啟右上角的「分享」選單。
![image](https://hackmd.io/_uploads/Sy2lBwDWkl.png =300x)
2. 按一下「更多功能」，確認筆記設定和建議修訂已「開啟」。
![image](https://hackmd.io/_uploads/HJdLHPPW1e.png =300x)
3. 這也能確認誰可以在您擁有的筆記上留言、編輯。

建議修訂相當直接。與建立留言一樣，若要建議修訂，請選取筆記中的一段文字，然後按一下「建議修訂」。

### 如何查看所有建議？
所有建議修訂都會出現在留言面板中。若要找到它，請按一下筆記右上方的 💬。點擊「留言」的下拉箭頭切換到「建議」。

這會顯示所有目前的建議，以及之前處理過的建議。

![image](https://hackmd.io/_uploads/S1C8-uDbyg.png =300x)

## 分享與設定權限
HackMD 專為即時協作而打造，因此有許多方式與他人分享您的工作。

使用筆記設定權限，確保您的合作者按照您希望的方式存取內容。

### 分享按鈕
按一下筆記右上角的「分享」按鈕。在這裡您可以
1. 邀請他人參與閱讀／協作您的筆記。
2. 查看並變更誰可以存取筆記，及他們擁有的權限等級。
3. 編輯筆記的固定網址。
4. 複製筆記的連結與他人分享。
5. 發表您的筆記。(稍後會深入討論這一點）。
![image](https://hackmd.io/_uploads/ByQh39r30.png =150x)

### 筆記連結 / 固定網址
在分享功能表的頂端是筆記的 URL 或連結。

在這裡您可以編輯連結，並透過複製來和他人分享。

### 筆記權限
設定完成後，您會發現下方有「筆記權限」區塊。您可以設定能閱讀筆記的人和能筆記中編輯的人。
![image](https://hackmd.io/_uploads/rkMbhur60.png =300x)

>[!Caution] 請留意
>一旦閱讀權限設定為「所有人」，閱讀權限將不再受限於筆記擁有者或已登入的讀者。只要得知連結，真的所有人都可以看到，請謹慎設定。

### 受邀者
在「分享」功能表的底部，您會找到「受邀者」區塊。您可以邀請個別 HackMD 使用者加入您的筆記 - **被邀請者需要有 HackMD 帳戶才能被邀請加入您的筆記**。
![image](https://hackmd.io/_uploads/rkna2cBnA.png =300x)

## 公開發表
只要幾秒鐘，立刻公開發表您的筆記！

1. 開啟「分享」選單並切換到「發佈」標籤。
2. 在公開發表之前，請確保您的筆記遵守 [HackMD 社群守則](https://hackmd.io/@hackmd/community-guidelines)！
3. 點擊「公開發佈筆記」
4. 成功發佈了！

![image](https://hackmd.io/_uploads/r1RsALHpA.png =300x)
